# Sales-Performance-Dashboard-Excel
An end-to-end data analysis project using Excel.

Project Overview
This project involved analyzing a dataset of 200 sales transactions to identify key performance drivers, regional trends, and customer behavior. The goal was to transform raw sales data into an interactive, executive-level dashboard that provides actionable business insights.

Key Insights
Revenue Leader: The East Region emerged as the top performer, contributing $1,506,250 to the total revenue.

Product Performance: Electronics is the dominant category, generating $1,764,403 in sales.

Sales Trend: January was the strongest month of the year with $962,330 in total sales.

Customer Loyalty: Identified 46 repeat customers out of a total of 50 unique customers, indicating a very high retention rate.

Transaction Efficiency: The Average Order Value (AOV) is $24,530.79, with UPI being the most preferred payment method (76 uses).

Tools & Skills
Excel: Advanced formulas (SUMIFS, VLOOKUP), Data Validation, and Pivot Tables.

Data Cleaning: Handled inconsistencies in date formats and categorical labels.

Data Visualization: Designed an interactive dashboard with Slicers, Donut Charts, and Trend Lines for executive reporting.
